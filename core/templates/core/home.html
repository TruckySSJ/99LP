Que es 99LP?

99LP es una web app que nos permite obtener datos estadisticos de los jugadores de la escena competitiva del videojuego League of Leguends.

Analisis

En 99LP encontraras analisis estaditicos de todos los jugadores.

Graficos

Encontraras graficos comparativos de las estadisticas de cada jugador.

Datos actualizados

Todos los datos se iran actualizando constantemente despues de cada fecha de partidas.

De donde saca los datos 99LP?

Los datos son scrappiados desde el sitio web Gol.gg para luego ser limpiados y almacenados en la Web App 99LP.

Como nace 99LP?

Esta Web App nace a travez de la necesidad de poder visualizar y comparar jugadores de la escena competitiva de League of Leguends de manera facil y rapida.

Tecnologias

Las tecnologias ocupadas para este proyecto fueron las siguientes: como lenguaje base Python, para la obstencion de datos BeatifoulSoup y para el despliegue el Framework Django.
